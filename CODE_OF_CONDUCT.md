# Code of Conduct

## Our Pledge

我们承诺为所有参与者提供友好、安全、包容的社区环境。

## Expected Behavior

- 尊重他人，理性讨论
- 聚焦问题本身，避免人身攻击
- 接受建设性反馈
- 维护开源协作氛围

## Unacceptable Behavior

- 侮辱、歧视、骚扰、威胁
- 恶意引战、刷屏、发布隐私信息
- 发布违法内容或明显侵犯他人权益的内容

## Enforcement

维护者有权对不当行为采取措施，包括但不限于警告、删除评论、限制参与权限。

## Scope

本规范适用于项目仓库、Issue、PR、讨论区及相关社区交流场景。
